# Currency converter (USD , SAR , EGY POUND)...This project was built to demonstrate core :- DOM Manipulation and Event handling concepts in javascript

## Features
real-time conversion ..updates the result instantly as the user types (using the `input` event)...built with pure vanilla javascript

## concepts applied:-
- selecting DOM elements using (document.getElementById).
- listening to user interactions via addEventListener("input",...).
- dynamically updating the UI text using (.innerHTML).
- Handling formatting and decimals using (.toFixed()).
